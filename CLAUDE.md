# frankdebruijn.com

Persoonlijke professionele website van Frank de Bruijn.

## Stack
- Astro (static site)
- Vanilla CSS (geen framework)
- Gehost op Cloudflare Pages — elke push naar `main` deployt automatisch

## Lokaal ontwikkelen
```bash
npm run dev      # start dev server op http://localhost:4321
npm run build    # build naar dist/
```

## Deployment
- GitHub: https://github.com/FrankdeBruijn/frankdebruijn-com
- Cloudflare Pages: https://frankdebruijn-com.pages.dev
- Live domein (nog te koppelen): https://frankdebruijn.com
